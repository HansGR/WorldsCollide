Final Fantasy 6 Menu Configurator

A web app to make and customize menu configuration files for patching FF6 Worlds Collide roms. See the project code for information on how to patch roms. Code by DoctorDT.

Custom window image

Upload a 32×32 image as the window's interior texture, pick a border treatment, and paint any tweaks. Everything you change here — the custom graphics, palette colors, and every menu setting above — is bundled into a single configuration file. Use the buttons at the bottom of the page to export it as ff6config.json (apply with ff6_config.py -i rom.smc --config ff6config.json) or to patch a ROM straight from your browser.

Click / drag in the canvas to paint with the selected palette color. Pick a brush size (1–8) and shape (square / circle), or switch to the Fill tool to flood-fill a region. The top 32×32 area is the interior texture; the bottom 32×24 area is the border / frame. Choose which window you're designing straight from the simulated menu at the top of the page (Page B → Window style), or use Export / Import .bmp to round-trip the 32×56 sheet through an external editor.

Export & patch

Export configuration (.json) bundles every menu setting and custom window image above into a single ff6config.json. Patch a ROM applies that same configuration to your ROM and returns the patched copy; the ROM must already be patched by Worlds Collide. Your ROM is sent to the server, patched in memory, and handed straight back — it is never stored.
